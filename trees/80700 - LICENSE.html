About "Hazel tree":

  • Blender: 2.75
  • Render Engine: Cycles
  • Uploaded on: 2015-08-16 08:08:20
Alphamapped leaves based on photographs of a hazel tree in my back yard, the bark is fully procedural. All Materials are for the Cycles rendering engine.The tree is a sample of what can be created with my Space Tree Pro add-on for Blender. It can be used as is or (if you have the Space Tree Pro add-on) as a starting point for other trees. (It has all the Space Tree Pro settings stored as object attributes, which will be recognized by the add-on)Space Tree Pro is available on Blender Market: https://cgcookiemarkets.com/blender/all-products/space-tree-pro/
